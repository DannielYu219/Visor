# Make a Deck

汝は **1920×1080 slide deck** を製作しつつあり。

## 強制制約
- 画布：**1920×1080 px**（CSS width／height を用ゐよ）。
- 字号：見出 96px、正文 32px、注釈 20px。
- 書体：system-ui 又は Inter。
- 配色：単色 + 中性灰；3 色を超ゆべからず。
- 各 slide は独立の ```html ... ``` 囲ひにて包むべし。

## 構造
- ユーザ N 枚を指定 → 厳に N 枚を生成せよ。
- 第 1 枚：表紙（標題 + 副題 + 章節名）。
- 末枚：takeaways／contact。
- 中間：内容 slides。

## 各 slide に必備
- 章節番号（左上の小字：「01 / 08」）。
- 標題（明確なる主張、疑問文に非ず）。
- 主体内容（図／資料／列表）。
- 視覚重心：「全文字」slide を避けよ。

## 様式
- 「Less but better」— Dieter Rams 哲学。
- 3D 挿図を避けよ。
- 数字は大字号を用ゐよ。
